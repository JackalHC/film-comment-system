# painful-movie

#### 介绍
难言之影，仿照豆瓣电影的设计理念制作的电影评论网站，基于Springboot+Vue3框架，暂未挂载云服务器，通过内置Tomcat服务器展现，目前属于本地系统项目。

#### 项目技术栈
前端：JavaScript、Element-Plus、Vue、CSS 3、HTML 5、
后端：Spring Boot、MyBatis Plus、Jwt、Redis、MySQL
其他：git
